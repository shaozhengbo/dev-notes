# 生产环境遇到的问题



1. 开发环境和生产环境读取resources目录下文件的路径问题（2020年12月24日）

   ```java
   // 开发环境
   this.getClass().getResource("/").getPath() + "/xxx/xxx.json"
   // 生产环境
   new ClassPathResource("/xxx/xxx.json")
   ```
