# mac 安装 nginx

```shell
brew install nginx
```



## Nginx配置文件路径

```
/usr/local/etc/nginx/nginx.conf
```



## Nginx安装目录

```
/usr/local/Cellar/nginx
```



## Nginx静态资源目录

```
/usr/local/var/www
```



## Nginx命令

```shell
# 启动
nginx

# 停止
nginx -s stop

# 重启
nginx -s reload

# 查看Nginx详细信息
nginx -V
```



## 线程pid记录文件

```
/usr/local/var/run/nginx.pid
```



## Nginx卸载

```shell
brew uninstall nginx
```
